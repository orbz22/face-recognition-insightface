# Folder ini akan berisi model InsightFace buffalo_l

Model tidak di-commit ke git karena ukurannya besar (~340MB).

## Download Model

Model akan otomatis di-download saat pertama kali menjalankan program.

Atau download manual dari:
https://github.com/deepinsight/insightface/releases/download/v0.7/buffalo_l.zip

Extract ke folder ini sehingga strukturnya:
```
models/
└── buffalo_l/
    ├── det_10g.onnx
    ├── w600k_r50.onnx
    ├── genderage.onnx
    ├── 1k3d68.onnx
    └── 2d106det.onnx
```
